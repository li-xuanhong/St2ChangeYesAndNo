# St2ChangeYesAndNo

[English](README.en.md)

让《杀戮尖塔 2》部分确认弹窗始终保持左侧为确认、右侧为取消。

## 功能

- 修改弃局确认弹窗。
- 修改断开连接确认弹窗。
- 修改主菜单“退出游戏”确认弹窗。
- 同步调整按钮语义、文案、回调与热键，而不只是单纯交换坐标。

## 效果

- 左侧按钮始终执行确认操作。
- 右侧按钮始终执行取消操作。
- 键盘或手柄上的确认 / 取消逻辑会和新的左右语义保持一致。

## 安装

1. 将 `St2ChangeYesAndNo.dll` 和 `St2ChangeYesAndNo.pck` 放入游戏 `mods` 目录。
2. 启动游戏。

## 兼容性

- 本模组基于 Slay the Spire 2 的标准 `.dll + .pck` 方式加载。
- 当前不需要额外安装 BaseLib。
