# St2ChangeYesAndNo

Keeps the confirm action on the left button and the cancel action on the right button for selected Slay the Spire 2 confirmation popups.

## Features

- Adjusts the abandon run confirmation popup.
- Adjusts the disconnect confirmation popup.
- Adjusts the main menu quit confirmation popup.
- Rebinds button meaning, text, callbacks, and hotkeys instead of only swapping positions.

## Result

- The left button always confirms.
- The right button always cancels.
- Keyboard and controller confirm / cancel behavior stays aligned with the new left / right semantics.

## Installation

1. Put `St2ChangeYesAndNo.dll` and `St2ChangeYesAndNo.pck` into the game's `mods` folder.
2. Launch the game.

## Compatibility

- This mod uses the standard Slay the Spire 2 `.dll + .pck` loading format.
- No extra BaseLib dependency is required at the moment.
